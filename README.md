# swift-hello-world
Hello World in Swift (iOS)

1) Download and Install Xcode
   https://developer.apple.com/xcode/

2) Create a new Xcode project
   New > Project

3) Set the Project Details
   Project Name: Hello World
   Interface: SwiftUI
   Lifecyce: SwiftUI App
   Language: Swift

4) Save Application to your Computer
   Choose a location to save the project and click, "Create"

5) Run the Application
   Press the Play button on the top left of the Xcode window
